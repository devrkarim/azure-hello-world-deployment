# Azure VM Static Website

This project demonstrates hosting a static HTML website on Microsoft Azure using a Virtual Machine.

## What I Did
- Created and configured a Virtual Machine in Microsoft Azure
- Connected to the VM using RDP/SSH
- Deployed a static HTML page to the VM
- Configured networking to make the site publicly accessible

## Technologies Used
- Microsoft Azure
- Virtual Machines
- HTML
- Remote Desktop / SSH

## Result
A live static website hosted on Azure and accessible via a public IP address.
